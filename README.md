# EEGConsciouslySeenUnseen
Scripts for analyses presented in the paper titled 'Neural representation of consciously seen and unseen information' (Open Access link: https://doi.org/10.1038/s41598-025-92490-y). 

Raw EEG data can be found in the OpenNeuro repository: https://openneuro.org/datasets/ds005273/versions/1.0.0.
